# Disclaimer 
Web scraping is delicate activity. it is recommended to be fair with the scraped website:
- respect the original website licences regarding its content : in the present case, original content is licenced "CC BY SA 4.5"
- do not stress the original website when scraping its pages : min 2s between two queries.
